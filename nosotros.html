Estampación screen (serigrafia)

La serigrafía es un proceso artesanal muy antiguo que se basa en la transmisión de tinta a través de una malla directamente sobre la prenda. La estampación se realiza color por color.

en nuestro taller disponemos de pulpos que permiten realizar el proceso de serigrafia de una forma eficiente, las tintas utilizadas para este proceso pueden ser del tipo plastisol o textil segun las necesidades del cliente

en serigrafia el numero maximos de tintas que se utlizaria ,seria de 5 tintas en prendas blancas y 4 tintas en prendas oscuras, este proceso es muy utlizado para la realizacion de prendas para moda

ventajas y desventajas

  • es un metodo de reproduccion de motivos en serie
  • gran calidad de estampacion
  • para un minimo de 20 prendas ya que de forma individual los precios serian muy altos
  • solo se estampa con tintas planas
  • Para hacer degradados sencillos, disponemos de la serigrafía con tramas.
  • la resistencia al lavado de las camisetas personalizadas en serigrafía es muy alta.
  • Para tiradas cortas, o si necesitás camisetas todas diferentes recomendamos la estampacion digital

Estampación Digital

vinilo

Este sistema de estampación funciona con láminas termoadhesivas que se adhieren a las prendas a través del calor. La estampación se realiza en 2 pasos: recortamos el diseño mediante un plotter de corte y después lo termofijamos sobre la prenda. Tenemos una gran variedad de colores y tonos metalizados. Muy recomendado para estampar dibujos y logos sencillos. A un solo color.Perfecto para hacer camisetas y dotaciones deportivas. Es muy resistente al lavado. Permite realizar tiradas muy cortas, desde una sola camiseta de forma rápida.

transfer

Permite estampar camisetas con foto o diseños con degradados y sombras, no aptos para serigrafía. Es uno de los mejores sistemas para hacer camisetas personalizadas muy baratas de forma rápida y con resultados a todo color y de alta calidad! Es el sistema más utilizado para Camisetas despedidas. Genial para estampar fotos divertidas y diseños vistosos.

Es un sistema de estampación textil. Se imprime un papel especial con tintas especiales para transfer.
